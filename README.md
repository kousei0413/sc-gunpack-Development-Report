# sc-gunpack-Development-Report

# まずHoldについて

#### ・Holdは待機アニメーションである。

#### ・aimのアニメーションと同時に再生することでエイムができるようになっている。

#### ・Holdとその他のアニメーション(ReloadやAimなど)を組み合わせることで動かしている。


# 発射(shoot)についての要点

#### ・実はスライドを過剰に後ろに下げている(この後でその相殺方法を説明する)

#### ・shootとholdを同時に再生するとスライドの後退が過剰ではなくなる。

#### ・aim時のレティクルは一時的に出現するようになっている。

####

# 豆知識
#### ・プレイヤーの向きについて
##### ↳Xは左右、Yは上下、Zは前後である。
###### ↳Xが+になると右、-なら左
######  Yが+になると上、-なら下
######  Zが+になると前、-なら後ろ
